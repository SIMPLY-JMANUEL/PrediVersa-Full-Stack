1️⃣  AWS BEDROCK (5 min)
   ┌─────────────────────────────────────────────────────────┐
   │ 1. Abre: https://console.aws.amazon.com/bedrock/        │
   │ 2. Menú → "Model Access"                                │
   │ 3. Busca "Meta Llama 3"                                 │
   │ 4. Haz clic: "Request Access" o "Enable"               │
   │ 5. Espera confirmación                                  │
   └─────────────────────────────────────────────────────────┘

2️⃣  CREDENCIALES AWS (5 min)
   ┌─────────────────────────────────────────────────────────┐
   │ 1. AWS Console → IAM → Users                            │
   │ 2. Security credentials → Create access key             │
   │ 3. Copia: Access Key ID                                 │
   │ 4. Copia: Secret Access Key                             │
   │ (Guarda en lugar seguro)                                │
   └─────────────────────────────────────────────────────────┘

3️⃣  CONFIGURAR BACKEND (10 min)
   ┌─────────────────────────────────────────────────────────┐
   │ cd backend                                              │
   │ cp .env.example .env                                    │
   │ # Editar .env y agregar:                                │
   │ AWS_REGION=us-east-1                                    │
   │ AWS_ACCESS_KEY_ID=tu_key_aqui                           │
   │ AWS_SECRET_ACCESS_KEY=tu_secret_aqui                    │
   │                                                         │
   │ npm install @aws-sdk/client-bedrock-runtime uuid        │
   │ npm start                                               │
   └─────────────────────────────────────────────────────────┘

POST /api/chatbot/message
├─ Descripción: Enviar mensaje a Llama 3
├─ Auth: JWT requerido
├─ Request body:
│  {
│    "message": "¿Cómo denuncio acoso?",
│    "context": { "topic": "violencia" }
│  }
├─ Response (200):
│  {
│    "success": true,
│    "data": {
│      "userMessage": "...",
│      "botResponse": "...",
│      "conversationId": "uuid"
│    }
│  }
└─ Errores: 400 (mensaje vacío), 401 (sin JWT), 500 (error AWS)

POST /api/chatbot/conversation
├─ Descripción: Crear nueva conversación
├─ Auth: JWT requerido
├─ Request body: { "topic": "bienestar_emocional" }
├─ Response: { "success": true, "data": { "id": "...", "topic": "..." } }
└─ Status: 201 Created

GET /api/chatbot/conversation/:conversationId
├─ Descripción: Obtener historial
├─ Auth: JWT requerido
├─ Response: { "success": true, "data": { "messages": [...] } }
└─ Status: 200 OK

DELETE /api/chatbot/conversation/:conversationId
├─ Descripción: Eliminar conversación
├─ Auth: JWT requerido
└─ Status: 200 OK

GET /api/chatbot/health
├─ Descripción: Verificar estado AWS Bedrock
├─ Auth: No requerido
├─ Response: { "success": true, "data": { "connected": true } }
└─ Status: 200 OK
